# Todo app

a fullstack todo app

## Tools

### back side:

```
Node js
GraphQL
express js
Apollo Server
Typescript
mongodb
```

### Front side:

```
React js
GraphQL
Apollo Client
Typescript
```

##### Run app:

First Step: install dependencies

```
cd todo-client && yarn
```

```
cd todo-server && yarn
```

Second Step :run back and fronT

```
Front : yarn start
```

```
back: yarn dev
```
